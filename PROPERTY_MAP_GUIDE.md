# Property Map Guide

## Overview
The PropertyMap component provides an interactive map for displaying and setting property locations with precise coordinates and address information.

## Features

### For Buyers/Viewers
- **Interactive Map Display**: See exact property location on OpenStreetMap
- **Marker Information**: Click marker to see property details and coordinates
- **Zoom Controls**: Zoom in/out to see surrounding area
- **Address Display**: View full address and GPS coordinates
- **Clear Location**: Understand property's position relative to city/landmarks

- **Search Address**: Search for property location by address
