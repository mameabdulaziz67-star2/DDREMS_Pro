================================================================================
                         ACTION BUTTONS & FUNCTIONALITY
================================================================================

CUSTOMER ACTIONS:
  1. 💳 Submit Payment
     - Select payment method (Bank Transfer, Cash, Check, Card)
     - Enter payment amount
     - Provide receipt file path
     - Sends notification to admin

  2. 📄 Upload Receipt
     - Upload receipt file path
     - Add receipt file name
     - Sends notification to admin

ADMIN ACTIONS:
  1. 📄 Generate Agreement
     - Select template ID
     - Generates agreement document
     - Sends notification to customer

  2. 📧 Send Agreement
     - Select recipient (Customer or Owner)
     - Sends agreement to selected party
     - Sends notification

  3. 🔔 Send Notification
     - Enter custom notification message
     - Sends to all parties
     - Tracked in notifications table

ALL USERS:
  1. 👁️ View Details
     - See full agreement information
     - View all details in modal

================================================================================
                         STATUS INDICATORS
================================================================================

Status Colors & Emojis:
  ⏳ Pending (Orange) - Waiting for action
  ➡️ Forwarded (Purple) - Forwarded to owner
  ✅ Accepted (Green) - Owner accepted
  ❌ Rejected (Red) - Owner rejected
  🎉 Completed (Green) - Transaction completed

Filter Tabs:
  📋 All - Show all agreements
  ⏳ Pending - Show pending only
  ✅ Accepted - Show accepted only
  🎉 Completed - Show completed only

================================================================================
                         DATABASE TABLES USED
================================================================================

agreement_requests
  - Main table with all agreement data
  - Stores customer, owner, property info
  - Tracks status and timestamps

agreement_payments
  - Stores payment information
  - Payment method, amount, date
  - Receipt file path and name

agreement_notifications
  - Stores all notifications
  - Recipient, type, message
  - Read status tracking
