┌─────────────────────────────────────────────────────────────────────────────┐
│                    BACKEND ROUTE STRUCTURE                                  │
└─────────────────────────────────────────────────────────────────────────────┘

/api/messages
│
├─ GET /user/:userId
│  └─ Fetch user's messages
│
├─ GET /unread/:userId
│  └─ Get unread count
│
├─ POST /
│  ├─ Middleware: verifyUser
│  ├─ Middleware: checkSendPermission
│  └─ Handler: Send single or group message
│
├─ PUT /read/:messageId
│  ├─ Middleware: verifyUser
│  └─ Handler: Mark message as read
│
├─ PUT /read-all/:userId
│  ├─ Middleware: verifyUser
│  └─ Handler: Mark all messages as read
│
├─ DELETE /:messageId
│  ├─ Middleware: verifyUser
│  └─ Handler: Delete message
│
└─ POST /bulk
   ├─ Middleware: verifyUser
   ├─ Middleware: checkSendPermission
   └─ Handler: Send bulk message to role
