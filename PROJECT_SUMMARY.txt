DIRE DAWA REAL ESTATE MANAGEMENT SYSTEM (DDREMS)
Admin Dashboard - Project Summary
================================================================================

PROJECT OVERVIEW
----------------
A modern, attractive, and fully functional admin dashboard for managing real 
estate operations in Dire Dawa. Built with React, Node.js, and MySQL.

TECHNOLOGY STACK
----------------
Frontend:
- React.js (v19.2.4)
- CSS3 (Custom styling with gradients and animations)
- Axios for API communication

Backend:
- Node.js with Express.js
- MySQL2 for database operations
- JWT for authentication
- Bcrypt for password hashing

Database:
- MySQL on WAMP Server (Port 3307)
- Database name: ddrems

DATABASE SCHEMA
---------------
Tables Created:
1. users - System users (admin, brokers, clients)
2. brokers - Real estate broker information
3. properties - Property listings
4. transactions - Property transactions
5. payments - Installment payment tracking
6. user_preferences - User property preferences
7. fraud_alerts - Fraud detection system

Sample Data Included:
- 1 Admin user (admin@ddrems.com / admin123)
- 3 Sample brokers
- 4 Sample properties (Villa, Apartment, Commercial, Land)

API ENDPOINTS
-------------
Authentication:
POST /api/auth/login - User login

Dashboard:
GET /api/dashboard/stats - Dashboard statistics
GET /api/dashboard/activities - Recent activities

Properties:
GET /api/properties - Get all properties
GET /api/properties/:id - Get property by ID
POST /api/properties - Create property
PUT /api/properties/:id - Update property
DELETE /api/properties/:id - Delete property

Brokers:
GET /api/brokers - Get all brokers
GET /api/brokers/:id - Get broker by ID
POST /api/brokers - Create broker
PUT /api/brokers/:id - Update broker

Users:
GET /api/users - Get all users

Transactions:
GET /api/transactions - Get all transactions
