⚠️ COMMON ISSUES & FIXES:

Issue: "Failed to send message"
Fix: 
  1. Check browser console (F12) for error details
  2. Verify userId is being passed correctly
  3. Ensure recipient exists in database
  4. Check user has permission to send messages

Issue: Messages not appearing in inbox
Fix:
  1. Verify message was sent successfully (check success message)
  2. Refresh the Messages page
  3. Check you're logged in as the recipient
  4. Verify message_recipients table has entries

Issue: Bulk message button not showing
Fix:
  1. Verify you're logged in as System Admin or Property Admin
  2. Check user role in database
  3. Restart backend server
