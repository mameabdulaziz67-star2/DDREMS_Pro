DATABASE STRUCTURE
================================================================================

MESSAGES TABLE:
- id (INT, PRIMARY KEY)
- sender_id (INT, FOREIGN KEY)
- receiver_id (INT, FOREIGN KEY, NULL for group)
- property_id (INT, FOREIGN KEY)
- subject (VARCHAR 255)
- message (TEXT)
- message_type (VARCHAR 50)
- status (ENUM: sent, delivered, read, failed)
- is_read (BOOLEAN) - for individual messages
- is_group (BOOLEAN) - 0 for individual, 1 for group
- updated_at (TIMESTAMP)
- created_at (TIMESTAMP)

MESSAGE_RECIPIENTS TABLE (for group messages):
- id (INT, PRIMARY KEY)
- message_id (INT, FOREIGN KEY)
- user_id (INT, FOREIGN KEY)
- is_read (BOOLEAN) - for group messages
- read_at (TIMESTAMP)
- created_at (TIMESTAMP)

NOTIFICATIONS TABLE:
- id (INT, PRIMARY KEY)
- user_id (INT, FOREIGN KEY)
- title (VARCHAR 255)
- message (TEXT)
- type (ENUM: info, success, warning, error, request)
- is_read (BOOLEAN)
- link (VARCHAR 500)
- action_url (VARCHAR 255)
- related_id (INT)
- created_at (TIMESTAMP)

NOTE: notification_type column has been REMOVED from notifications table

HOW TO USE
================================================================================

SENDING INDIVIDUAL MESSAGE:
1. Go to Send Message
2. Select single user
3. Enter subject and message
4. Click Send
5. Message appears in recipient's inbox
6. Notification created for recipient

SENDING GROUP MESSAGE:
1. Go to Send Message
2. Select multiple users
3. Enter subject and message
4. Click Send
5. Message appears in all recipients' inboxes
6. Notifications created for all recipients

VIEWING MESSAGES:
1. Go to Messages
2. All messages displayed (individual + group)
3. Shows sender, subject, date
4. Shows read/unread status
5. Shows recipient count for group messages

MARKING AS READ:
1. Click on message
2. Message marked as read
3. Unread count decreases
4. Notification marked as read

DELETING MESSAGE:
1. Click delete button on message
2. Message deleted from inbox
3. For group messages, deleted from all recipients

EDITING MESSAGE:
1. Click edit button (sender only)
2. Edit subject and message
3. Click Save
4. Message updated
5. Edit history can be tracked
