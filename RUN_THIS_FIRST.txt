🚀 DEPLOYMENT STEPS (Copy and paste these commands):

STEP 1: Backup Database
────────────────────────
mysqldump -u root -p ddrems > ddrems_backup.sql


STEP 2: Apply Database Migration
─────────────────────────────────
mysql -u root -p ddrems < database/add-missing-tables.sql


STEP 3: Kill Existing Node Process
───────────────────────────────────
taskkill /F /IM node.exe


STEP 4: Start Backend Server
─────────────────────────────
node server/index.js

(You should see: "Server running on port 5000")


STEP 5: In Another Terminal - Start Frontend
──────────────────────────────────────────────
cd client
npm start

(You should see: "Compiled successfully!")


STEP 6: Test Endpoints
──────────────────────
curl http://localhost:5000/api/brokers
curl http://localhost:5000/api/messages/user/1
curl http://localhost:5000/api/profiles/pending

🚨 TROUBLESHOOTING:

If something doesn't work:

1. Check database migration:
   mysql -u root -p ddrems -e "SHOW TABLES;"

2. Check backend is running:
   curl http://localhost:5000/api/brokers

3. Check frontend is running:
   Open http://localhost:3000 in browser

4. Check server logs:
   Look at the terminal where you ran "node server/index.js"

5. Kill port 5000 if needed:
   taskkill /F /IM node.exe
