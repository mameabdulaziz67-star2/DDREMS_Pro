🚀 HOW TO USE
═════════════════════════════════════════════════════════════════════════════

SEND SINGLE MESSAGE:
  1. Login as any user
  2. Go to "Send Message"
  3. Select "Single User" mode
  4. Choose recipient
  5. Fill subject and message
  6. Click "Send Message"
  7. Recipient will see message in inbox

SEND GROUP MESSAGE:
  1. Login as any user
  2. Go to "Send Message"
  3. Select "Group Message" mode
  4. Select multiple recipients
  5. Fill subject and message
  6. Click "Send Message"
  7. All recipients will see message

SEND BULK MESSAGE (Admin Only):
  1. Login as System Admin or Property Admin
  2. Go to "Send Message"
  3. Select "Bulk (By Role)" mode
  4. Choose role (e.g., "All Owners")
  5. Fill subject and message
  6. Click "Send Message"
  7. All users of that role will see message

RECEIVE MESSAGES:
  1. Login as any user
  2. Go to "Messages"
  3. See all messages sent to you
  4. Click on message to read
  5. Message automatically marked as read
