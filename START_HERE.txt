🔧 IF SOMETHING DOESN'T WORK
═══════════════════════════════════════════════════════════════

Problem: "Port 5000 already in use"
Solution: 
  1. Close all Command Prompt/PowerShell windows
  2. Open Task Manager (Ctrl+Shift+Esc)
  3. End all "Node.js" processes
  4. Wait 10 seconds
  5. Try starting again
  
  OR read: FIX_PORT_IN_USE_ERROR.txt

Problem: Server won't start
Solution: Check WAMP is running, verify .env file

Problem: Can't login
Solution: Password is admin123 for all accounts

Problem: No properties showing
Solution: Add properties as owner, approve as admin

Problem: Images not displaying
Solution: Run: node apply-database-fixes.js
