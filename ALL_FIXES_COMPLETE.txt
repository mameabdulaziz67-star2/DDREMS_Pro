Problem: No properties showing
Solution:
  1. Login as owner/broker
  2. Add some properties
  3. Login as admin
  4. Approve properties (set status to 'active')
  5. Refresh customer dashboard

Problem: Images not displaying
Solution:
  1. Run: mysql -u root -P 3307 ddrems < COMPLETE_FIX_ALL_ISSUES.sql
  2. Restart servers
  3. Clear browser cache (Ctrl+Shift+Delete)

Problem: Users can't login
Solution:
  1. Run: node add-missing-users.js
  2. Check if user exists in database
  3. Password is: admin123

Problem: Database connection error
Solution:
  1. Make sure WAMP is running (green icon)
  2. Check .env file:
     DB_HOST=localhost
     DB_PORT=3307
     DB_NAME=ddrems
     DB_USER=root
     DB_PASSWORD=

Problem: Port already in use
Solution:
  1. Kill existing processes: taskkill /F /IM node.exe
  2. Wait 5 seconds
  3. Start servers again
