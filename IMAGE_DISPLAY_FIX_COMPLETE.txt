TECHNICAL DETAILS:
================================================================================

Image Storage:
- Field: property_images.image_url (LONGTEXT)
- Format: data:image/png;base64,iVBORw0KGgo...
- Size: Varies (test images are 126 bytes, real images will be larger)

Image Retrieval:
- API: GET /api/properties
- Returns: main_image field with base64 data
- Frontend: Displays directly in <img src={main_image} />

Image Upload Flow:
1. User selects image files
2. FileReader converts to base64
3. POST to /api/property-images with base64 data
4. Stored in database LONGTEXT field
5. Retrieved and displayed in dashboards
