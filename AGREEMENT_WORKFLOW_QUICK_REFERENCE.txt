================================================================================
                         10-STEP WORKFLOW OVERVIEW
================================================================================

STEP 1: CUSTOMER REQUEST
  Action: Customer clicks "Request Agreement" on dashboard
  Status: pending_admin_review
  Notification: Property admin receives notification

STEP 2: ADMIN FORWARD
  Action: Property admin clicks "Forward to Owner"
  Status: waiting_owner_response
  Notification: Owner receives notification

STEP 3: OWNER DECISION
  Action: Owner clicks "Accept" or "Reject"
  Status: owner_accepted or owner_rejected
  Notification: Admin receives notification

STEP 4: GENERATE AGREEMENT ⭐ AUTO-POPULATE TRIGGERED HERE
  Action: Admin clicks "Generate Agreement"
  Status: agreement_generated
  Auto-Populate: 32 fields populated from profiles & documents
  Notification: Customer receives notification

STEP 5: CUSTOMER EDIT
  Action: Customer clicks "Edit Fields"
  Fields: Auto-populated fields displayed
  Read-Only: Profile, property, broker, agreement fields (🔒)
  Editable: User-specific fields
  Tracking: Changes tracked with user ID & timestamp

STEP 6: CUSTOMER SUBMIT
  Action: Customer clicks "Submit with Payment"
  Payment: Method, amount, receipt file path
  Status: customer_submitted
  Notification: Admin receives notification

STEP 7: ADMIN REVIEW
  Action: Admin clicks "Review Submission"
  Decision: Approve, Reject, or Suspend
  Status: waiting_owner_final_review (if approved)
  Notification: Owner receives notification

STEP 8-9: OWNER FINAL REVIEW
  Action: Owner clicks "Submit Final Review"
  Notes: Owner can add final notes
  Status: owner_submitted
  Notification: Admin receives notification

STEP 10: COMMISSION & HANDSHAKE
  Action 1: Admin clicks "Calculate Commission"
    - Commission: 5% customer + 5% owner
    - Status: ready_for_handshake
  Action 2: Both parties click "Final Handshake"
    - Transaction created
    - Status: completed
    - Notifications sent to both parties

================================================================================
                         COMMISSION CALCULATION
================================================================================

Formula:
  Customer Commission = (Property Price × Commission %) / 100
  Owner Commission = (Property Price × Commission %) / 100
  Total Commission = Customer + Owner

Example:
  Property Price: 100,000 ETB
  Commission Rate: 5%
  Customer Commission: 5,000 ETB
  Owner Commission: 5,000 ETB
  Total Commission: 10,000 ETB

================================================================================
                         TROUBLESHOOTING
================================================================================

Issue: Auto-populate fields not showing
Solution: Ensure customer/owner profiles are complete with all information

Issue: Commission calculation incorrect
Solution: Verify commission_percentage is set correctly (default 5%)

Issue: Notifications not received
Solution: Check agreement_notifications table for records

Issue: Fields not editable
Solution: Verify field is_editable flag is set to true

Issue: Status not updating
Solution: Check agreement_workflow_history for action logs
